# 🏷️ Label Padega Sabh

**Label Padega Sabh** is an AI-powered consumer product intelligence platform that helps users make informed purchasing decisions by analyzing packaged food products through barcode scanning or label image recognition. The platform simplifies complex ingredient labels into easy-to-understand insights, helping consumers understand what they are buying and consuming.

## ✨ Features

- 📷 Barcode scanning using the device camera
- 📝 OCR-based ingredient extraction from product labels
- 🧠 AI-powered ingredient explanations using Gemini
- 🌍 Global regulatory comparison across multiple countries
- ⚠️ Personalized health warnings based on user profile
- 📊 Product health and concern score
- 📰 Food safety news and regulatory updates
- 🔍 Product comparison and healthier alternatives
- 📚 Scan history and user dashboard

## 🛠️ Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Python, FastAPI
- **Database:** PostgreSQL, MongoDB, Redis
- **AI:** Google Gemini API
- **Data Sources:** OpenFoodFacts, USDA FoodData Central

## 🚀 Getting Started


Navigate to the project directory:

```bash
cd label_pedega_sabh
```

Install dependencies and start the application by following the setup instructions for the frontend and backend.

## 🎯 Project Goal

Our goal is to make food labels transparent and understandable for everyone. Instead of reading complex ingredient names and technical information, users receive simple explanations, health insights, regulatory information, and personalized recommendations to make smarter purchasing decisions.
